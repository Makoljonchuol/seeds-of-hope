# Ubuntu House Kakuma

A responsive web application showcasing a community-led humanitarian initiative supporting refugee children in Kakuma Refugee Camp, Kenya.

## Project Overview

Ubuntu House Kakuma is a web platform built to raise awareness and support for 11 refugee students in Kakuma Refugee Camp. The application provides information about the initiative, individual student profiles, and ways to contribute.

**Live Demo:** [Add your deployed link here]

## Mission

Supporting refugee children with:
- Daily food security when aid falls short
- Educational support and mentorship
- Safe housing and community care
